# 🚀 MultiTool Dashboard

A beautifully designed, premium web-based multi-tool workspace built with pure HTML, CSS (Glassmorphism), and Vanilla JavaScript. 

This dashboard serves as a unified digital workshop for developers, students, and professionals. It features over 30 distinct tools—ranging from programming formatters and advanced matrix calculators to QR generators—all overseen by a state-of-the-art, dual-engine AI assistant with an Apple VisionOS-inspired interface.

---

## ✨ The MultiTool AI Assistant

The crown jewel of this dashboard is **MultiTool**, a fully integrated, context-aware AI assistant built with a premium, sci-fi-inspired UI and an indestructible backend.

### 🧠 Dual-Engine Architecture (Zero Downtime)
* **Primary Brain (Google Gemini):** Powered by the lightning-fast `gemini-3.5-flash` model for intelligent, context-aware conversations and code generation.
* **Auto-Fallback Engine (Groq / Llama OSS):** If Gemini hits a rate limit or a network error occurs, the system instantly and silently hands the conversation history over to Groq's `openai/gpt-oss-20b` LPU network, ensuring the AI never goes offline.

### 🎨 Apple VisionOS-Inspired Interface
* **Magnetic Energy Orb:** A floating, glowing glass sphere with rotating dashed rings and orbiting particles that physically reaches for your cursor (Cursor Magnetism).
* **Dynamic Mood States:** The AI visually reacts to the conversation. The orb glows Blue when idle, shifts to Purple and spins faster when "thinking", and pulses on success.
* **Materializing Messages:** Chat bubbles don't just appear; they materialize with a cinematic blur-to-sharp scaling animation.
* **Cursor Spotlight:** A subtle, premium radial gradient tracks your mouse movements across the chat glass panel.

### 🎵 Custom Web Audio Synthesizer
Zero external audio files. The dashboard uses the browser's native Web Audio API (`AudioContext`) to generate sci-fi hums, digital pings, and hover clicks in real-time.

### 👾 Easter Eggs & Persona
The AI is programmed with a witty, chill, and slightly sarcastic developer persona. It randomly generates contextual thought bubbles while idle and features hidden interactions (try spam-clicking the orb!).

## 🛠️ The Tool Arsenal

The dashboard contains over 30 modular tools divided into intuitive categories:

### 💻 Programming & Developer Tools
* **JSON Suite:** Formatter, Validator, and interactive Tree Viewer.
* **XML & YAML:** Safely parses, formats, and beautifies markup.
* **Web Languages:** HTML, CSS, and Vanilla JS auto-formatters.
* **Encoders:** Base64 Encoder/Decoder.

### 🧮 Advanced Mathematics
* **Calculators:** Standard, Scientific, Programmer, and Graphing calculators.
* **Algebra & Logic:** Matrix Calculator, Complex Number Solver, Expression Evaluator.
* **Number Theory:** Prime Number Checker, GCF & LCM Calculators.
* **Fractions & Ratios:** Fraction solver, Percentage Calculator, Proportion Calculator.

### 📊 Data & Statistics
* Instantly process massive datasets with the **Average, Mean, Median, and Mode** calculators (including bimodal dataset detection).

### ⚙️ Utilities & Generators
* **Generators:** QR Code Generator, standard CODE128 Barcode Generator, Random Number Generator.
* **Life & Productivity:** Date & Time utilities, Unit Converters, Financial calculators, and Health & Fitness trackers.
* **Design:** Advanced Color Picker (HEX, RGB, HSL extraction).

## 🎨 UI / UX Highlights

* **Premium Glassmorphism:** Deep frosted glass panels, neon accents, and smooth transitions built purely with CSS custom variables.
* **Intelligent Search:** A smart search bar that instantly filters through the 30+ tools and dynamically opens/closes dropdown folders based on matching results.
* **Instant Copying:** 1-click clipboard integration across all code formatters, color pickers, and math outputs.
* **Staggered Animations:** Navigation buttons and tool cards slide in smoothly on page load using CSS keyframes.
